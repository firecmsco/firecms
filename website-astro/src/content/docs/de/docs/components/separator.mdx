Trennlinien (Separators) werden verwendet, um Inhalte in einem Layout oder einer Liste visuell voneinander abzugrenzen. Sie können entweder horizontal oder vertikal sein, was sie für verschiedene Designanforderungen vielseitig macht.

## Verwendung

Um `Separator` zu verwenden, importieren Sie ihn aus Ihren Komponenten und geben Sie die Eigenschaft `orientation` an und optional die Eigenschaft `decorative`, um sein Erscheinungsbild zu steuern.
