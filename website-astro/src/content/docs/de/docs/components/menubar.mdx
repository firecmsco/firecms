Die Menubar-Komponente bietet eine anpassbare und barrierefreie Menüleiste, die mit den Menubar-Primitiven von Radix UI erstellt wurde. Sie kann verwendet werden, um komplexe Menüstrukturen mit verschiedenen Funktionen wie Untermenüs, Kontrollkästchen und Optionsgruppen (Radio Groups) zu erstellen.

## Verwendung

Um `Menubar` zu verwenden, importieren Sie es aus Ihren Komponenten und setzen Sie es mit anderen zugehörigen Menubar-Komponenten wie `MenubarMenu`, `MenubarTrigger`, `MenubarContent`, `MenubarItem` usw. zusammen.
