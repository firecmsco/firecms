Beim Erstellen einer Benutzeroberfläche ist es oft notwendig, Icons einzubinden, um verschiedene Aktionen, Zustände oder Inhalte darzustellen.
FireCMS verwendet **Material Design Icons**, die kostenlos von Google erhältlich sind. Diese Icons sind praktischerweise
als unabhängige React-Komponenten verpackt, die in Ihren eigenen Komponenten verwendet werden können.

In manchen Fällen müssen Sie die Icons als Komponenten verwenden, zum Beispiel in einem Button. In anderen Fällen müssen Sie möglicherweise
den Icon-Schlüssel angeben.

In dieser Ansicht können Sie alle verfügbaren Icons und ihre Schlüssel sehen. **Suchen** Sie einfach nach dem gewünschten Icon!
