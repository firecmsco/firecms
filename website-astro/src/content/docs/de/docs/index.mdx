:::important
FireCMS ist ein **Open-Source-Headless-CMS- und Admin-Panel-Framework**, das mit React und TypeScript entwickelt wurde. Es verbindet sich direkt mit Ihrem **Firebase**- oder **MongoDB**-Projekt und bietet eine leistungsstarke, intuitive und typsichere Oberfläche zur Verwaltung Ihrer Daten.
:::

FireCMS ist darauf ausgelegt, der schnellste Weg zu einem produktionsbereiten Back-Office zu sein. Es generiert vollständige CRUD-Ansichten (Erstellen, Lesen, Aktualisieren, Löschen) aus Ihrer Datenkonfiguration, legt **keine Datanstruktur-Einschränkungen** fest und kann in Minuten eingerichtet werden.

## Zwei Wege zum Einstieg

Der wichtigste erste Schritt ist die Entscheidung, welche Version von FireCMS die Richtige für Sie ist.

### FireCMS Cloud

Für Geschwindigkeit und Komfort. Eine vollständig verwaltete, wartungsfreie Lösung mit einer 1-monatigen kostenlosen Testphase.

  * ✅ **Starten Sie in 60 Sekunden**, keine Programmierung erforderlich.
  * ✅ Vollständig verwaltete Plattform mit automatischen Updates.
  * ✅ No-Code-Schema-Editor und Datenschemas-Inferenz.
  * ✅ Low-Code-Anpassung mit React-Komponenten.

[**Kostenlose Testphase starten**](https://app.firecms.co)

### Self-Hosted

Für maximale Kontrolle und Anpassung. Kostenlos, Open-Source und in Ihrer eigenen Umgebung gehostet.

  * ✅ **Kostenlos & Open Source** (MIT-Lizenz).
  * ✅ Volle Kontrolle über Ihren Code und Ihr Deployment.
  * ✅ Tiefe Anpassung mit React und TypeScript.
  * ✅ `npx create-firecms-app` zum Starten.

[**Self-Hosted Quickstart**](/docs/self)

## Was können Sie bauen?

FireCMS ist flexibel genug, um das Backend für jedes Projekt zu sein. Häufige Anwendungsfälle sind:

  * **Benutzerdefinierte Admin-Panels:** Erstellen Sie eine typsichere UI zur Verwaltung Ihrer Anwendungsdaten (z. B. Benutzer, Produkte, Bestellungen) ohne Frontend-Aufwand.
  * **Interne Tools:** Erstellen Sie maßgeschneiderte Tools für Ihre Betriebs-, Vertriebs- oder Support-Teams zur Verwaltung von Arbeitsabläufen, Datenverfolgung und internen Prozessen.
  * **Headless CMS:** Versorgen Sie Ihre Websites, mobilen Apps und Marketing-Inhalte über eine einzige leistungsstarke Admin-Oberfläche.

## Kern-Entwicklerfunktionen

FireCMS ist von Entwicklern für Entwickler gebaut.

### 1\. Schema als Code

Definieren Sie Ihre Datenmodelle als TypeScript-Objekte. Ihr Code wird zur einzigen Wahrheitsquelle und ermöglicht Versionskontrolle, Typsicherheit und einen nahtlosen Git-basierten Workflow.

### 2\. Sofortige CRUD-UI

FireCMS liest Ihr Schema und generiert automatisch eine vollständige, leistungsstarke und schöne CRUD-Oberfläche mit Tabellen, Formularen, Validierung und mehr.

### 3\. Radikale Erweiterbarkeit

Wenn Sie es in React bauen können, können Sie es in FireCMS bauen. Erstellen Sie benutzerdefinierte Formularfelder, völlig neue Top-Level-Ansichten oder ersetzen Sie jeden Teil der UI durch Ihre eigenen Komponenten. Keine Black Boxes.
