:::tip
FireCMS Cloud ist das **ultimative Firebase Admin-Panel** und **Headless CMS**.
Es bietet ein vollständig verwaltetes, **React-basiertes Back-Office**, mit dem Sie Ihre Firestore-Daten, Benutzer und Inhalte einfach verwalten können.

**No-Code bereit**: Daten sofort visualisieren und bearbeiten.
**Code-fähig**: Benutzerdefinierten React-Code, Entity-Ansichten und Property-Builder hochladen.
:::

## Warum FireCMS Cloud?

Ein benutzerdefiniertes **Admin-Panel** oder **internes Tool** von Grund auf zu erstellen ist zeitaufwändig. FireCMS Cloud gibt Ihnen ein produktionsbereites **CMS** in Minuten, maßgeschneidert für **Firebase** und **Firestore**.

### Hauptfunktionen

- **Verwalteter Service**: Wir kümmern uns um Hosting, Skalierbarkeit und Updates. Sie konzentrieren sich auf Ihre Daten und Geschäftslogik.
- **Schöne UI**: Eine moderne, responsive Oberfläche für Ihre **Firestore-Datenbank**.
- **Kollektions-Editor**: Definieren Sie Ihr Schema visuell mit unserem No-Code-Editor.
- **Entwicklerfreundlich**: Erweitern Sie mit **React**. Erstellen Sie benutzerdefinierte Formularfelder, Dashboard-Widgets und Vorschauen.
- **Rollenbasierter Zugriff**: Granulare Berechtigungssteuerung für Ihr Content-Team und Administratoren.


## Erste Schritte

Sie können FireCMS Cloud mit Ihrem vorhandenen Firestore-Projekt verwenden oder ein neues für Sie erstellen lassen.

Melden Sie sich einfach an und legen Sie los!
