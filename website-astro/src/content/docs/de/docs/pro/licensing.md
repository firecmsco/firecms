:::tip
Haben Sie Fragen, möchten eine Testversion anfordern oder benötigen eine benutzerdefinierte Lizenz?
Bitte [kontaktieren Sie uns per E-Mail](mailto:hello@firecms.co)
oder [planen Sie einen Anruf](https://calendar.google.com/calendar/u/0/appointments/schedules/AcZssZ0INW8ihjQ90S4gkdo8_rbL_Zx7gagZShLIpHyW43zDXkQDPole6a1coo1sT2O6Gl05X8lxFDlp?gv=true).
:::

Sie können die FireCMS PRO-Funktionen lokal während der Entwicklung ohne Lizenz nutzen, aber für das Deployment benötigen Sie eine.

Sie können eine Lizenz auf der [FireCMS Abonnements-Seite](https://app.firecms.co/subscriptions) erwerben.

Beim Erstellen Ihrer Lizenz müssen Sie die Projekt-IDs angeben, die die Lizenz verwenden werden. Ihre Projekt-ID finden Sie in der Firebase-Konsole.

Sie erhalten einen API-Schlüssel, den Sie an Ihre `FireCMS`-Komponente übergeben müssen.
Wenn Sie eine Starter-Vorlage verwenden, können Sie ihn in der `.env`-Datei festlegen.
