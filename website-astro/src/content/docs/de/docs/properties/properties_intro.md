---

Eigenschaften definieren jedes **Feld** in einem Formular oder eine Spalte in einer Kollektion innerhalb einer
Entity-Kollektion.

Sie können Eigenschaften erstellen, indem Sie das Objekt direkt erstellen oder indem Sie
die Hilfsmethode `buildProperty` verwenden (eine Identity-Funktion, die das
TypeScript-Typsystem zur Validierung der Eingabe verwendet).

Sie können auch Eigenschaften **dynamisch** aktualisieren, basierend auf der entityId, dem
Pfad oder den aktuellen Werten. Lesen Sie den
[Abschnitt über bedingte Felder](conditional_fields).

Lesen Sie über die verschiedenen verfügbaren Felder im Felder-Abschnitt oder die
verschiedenen Eigenschaftskonfigurationen.
