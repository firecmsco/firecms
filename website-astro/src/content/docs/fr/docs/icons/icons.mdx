Lors de la création d'une interface utilisateur, il est souvent nécessaire d'inclure des icônes pour représenter différentes actions, états ou contenus.
FireCMS utilise les **icônes Material Design**, disponibles gratuitement auprès de Google. Ces icônes sont regroupées
sous forme de composants React indépendants, que vous pouvez utiliser dans vos propres composants.

Dans certains cas, vous pourriez avoir besoin d'utiliser les icônes comme composants, par exemple dans un bouton. Dans d'autres cas, vous pourriez
avoir besoin de spécifier la clé de l'icône.

Dans cette vue, vous pouvez voir toutes les icônes disponibles et leurs clés. Il vous suffit de **rechercher** l'icône dont vous avez besoin !
