InfoLabel est un composant qui affiche un libellé avec une icône d'information. Il est couramment utilisé pour fournir des informations supplémentaires ou des conseils aux utilisateurs.

## Utilisation

Importez `InfoLabel` depuis `@firecms/ui` et fournissez le contenu comme enfant ou comme prop `label`.
