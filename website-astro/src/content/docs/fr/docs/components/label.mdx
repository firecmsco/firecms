Label est un composant qui affiche un libellé textuel. Il est couramment utilisé pour étiqueter les champs de formulaire, les éléments de liste et d'autres éléments d'interface.

## Utilisation

Importez `Label` depuis `@firecms/ui` et passez le contenu comme enfant.
