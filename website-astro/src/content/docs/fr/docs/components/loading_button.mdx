LoadingButton est un composant bouton qui affiche un indicateur de chargement lorsqu'une opération est en cours. Il est utile pour les actions asynchrones, comme les soumissions de formulaires ou les appels d'API.

## Utilisation

Importez `LoadingButton` depuis `@firecms/ui` et utilisez la prop `loading` pour contrôler l'état de chargement.
