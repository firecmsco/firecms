CircularProgress est un composant qui affiche un indicateur de chargement circulaire. Il est utilisé pour indiquer qu'une opération est en cours.

## Utilisation

Importez `CircularProgress` depuis `@firecms/ui` et utilisez-le pour afficher un indicateur de chargement.
