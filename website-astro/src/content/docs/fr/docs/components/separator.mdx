Separator est un composant de ligne de séparation visuelle. Il peut être rendu horizontalement ou verticalement pour diviser du contenu.

## Utilisation

Importez `Separator` depuis `@firecms/ui`. Utilisez la prop `orientation` pour contrôler la direction.
