:::important
FireCMS est un **framework de CMS headless et panneau d'administration open source** construit avec React et TypeScript. Il se connecte directement à votre projet **Firebase** ou **MongoDB** pour fournir une interface puissante, intuitive et typée pour gérer vos données.
:::

FireCMS est conçu pour être le moyen le plus rapide de créer un back-office prêt pour la production. Il génère des vues CRUD complètes (Créer, Lire, Mettre à jour, Supprimer) à partir de votre configuration de données, **n'impose aucune restriction de structure de données** et peut être configuré en quelques minutes.

## Que pouvez-vous créer ?

FireCMS est suffisamment flexible pour être le backend de n'importe quel projet. Les cas d'utilisation courants incluent :

  * **Panneaux d'administration personnalisés :** Créez une interface typée pour gérer les données de votre application (ex. : utilisateurs, produits, commandes) sans la complexité du frontend.
  * **Outils internes :** Créez des outils sur mesure pour vos équipes d'exploitation, de vente ou de support afin de gérer les flux de travail, suivre les données et exécuter des processus internes.
  * **CMS Headless :** Alimentez vos sites web, applications mobiles et contenus marketing depuis une seule interface d'administration puissante.

## Fonctionnalités principales pour les développeurs

FireCMS est conçu par des développeurs, pour des développeurs.

### 1\. Le schéma comme code

Définissez vos modèles de données en tant qu'objets TypeScript. Votre code devient la source de vérité unique, permettant le contrôle de version, la sécurité des types et un flux de travail basé sur git sans friction.

### 2\. Interface CRUD instantanée

FireCMS lit votre schéma et génère automatiquement une interface CRUD complète, performante et élégante avec des tableaux, des formulaires, une validation et bien plus encore.

### 3\. Extensibilité radicale

Si vous pouvez le construire avec React, vous pouvez le construire dans FireCMS. Créez des champs de formulaire personnalisés, de nouvelles vues de niveau supérieur, ou remplacez n'importe quelle partie de l'interface avec vos propres composants. Aucune boîte noire.
