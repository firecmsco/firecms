:::tip
FireCMS Cloud est le **panneau d'administration Firebase ultime** et le **CMS headless**.
Il fournit un **back-office basé sur React**, entièrement géré, qui vous permet de gérer vos données Firestore, vos utilisateurs et votre contenu facilement.

**Prêt sans code** : Visualisez et modifiez les données immédiatement.
**Code activé** : Téléversez du code React personnalisé, des vues d'entités et des constructeurs de propriétés.
:::

## Pourquoi FireCMS Cloud ?

Créer un **panneau d'administration** ou un **outil interne** personnalisé de zéro prend du temps. FireCMS Cloud vous donne un **CMS** prêt pour la production en quelques minutes, adapté à **Firebase** et **Firestore**.

### Fonctionnalités clés

- **Service géré** : Nous gérons l'hébergement, la scalabilité et les mises à jour. Vous vous concentrez sur vos données et votre logique métier.
- **Belle interface** : Une interface moderne et réactive pour votre **base de données Firestore**.
- **Éditeur de collection** : Définissez votre schéma visuellement avec notre éditeur sans code.
- **Adapté aux développeurs** : Étendez avec **React**. Créez des champs de formulaire personnalisés, des widgets de tableau de bord et des aperçus.
- **Accès basé sur les rôles** : Contrôle d'autorisation granulaire pour votre équipe de contenu et les administrateurs.
