Les propriétés définissent chaque **champ** dans un formulaire ou une colonne dans une collection incluse dans une collection d'entités.

Vous pouvez construire des propriétés en créant l'objet directement ou en
utilisant la méthode d'assistance `buildProperty` (juste la fonction d'identité qui utilise
le système de types TypeScript pour valider l'entrée).

Vous voudrez peut-être aussi mettre à jour les propriétés **dynamiquement**, en fonction de l'entityId, du
chemin ou des valeurs actuelles. Consultez
la [section des champs conditionnels](conditional_fields)

Consultez les différents champs disponibles dans la section des champs, ou les
différentes configurations de propriétés.
