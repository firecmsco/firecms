Build your own **Postgres admin panel** and **headless CMS** with full control over your infrastructure. Rebase lets you deploy a powerful **React-based back-office** on your own servers, with complete data ownership.

### Create a new project using the CLI

To create a new project using the CLI, you can run the following command:
```bash
npx create-rebase-app
```
or:

```bash
yarn create rebase-app
```


This will create a new Rebase project in the selected folder.
A Rebase project includes three parts:

- **`frontend/`** — React application using Rebase with the PostgreSQL data source
- **`backend/`** — Node.js server with PostgreSQL, Drizzle ORM, and WebSocket support
- **`shared/`** — Shared TypeScript collections used by both frontend and backend

### Prerequisites

- Node.js 18+
- PostgreSQL database
- pnpm (recommended)

### Environment Setup

After creating the project, configure your database connection in the `.env` file:

```bash
DATABASE_URL=postgresql://username:password@localhost:5432/your_database
```

### Running the project

Start both frontend and backend with a single command:

```bash
pnpm dev
```

This will:
- Start the backend server on `http://localhost:3001`
- Start the frontend development server
- Enable hot reloading for both applications
- Provide real-time WebSocket synchronization

You can also start them individually:

```bash
# Backend only
pnpm dev:backend

# Frontend only
pnpm dev:frontend
```

### Database operations

Rebase uses Drizzle ORM for type-safe database operations. Common database commands:

```bash
pnpm db:migrate      # Run database migrations
pnpm db:studio       # Open Drizzle Studio
pnpm generate:schema # Generate Postgres schema from your collections
```

### Changing the base URL of Rebase

Rebase uses React Router to handle the routing of the application.
You can change the base URL of Rebase by modifying the `basename` property of the `BrowserRouter` component in the `src/main.tsx` file.
For example, if you want to change the base URL to `/admin`, you can do it like this:

```tsx
import React from "react"
import ReactDOM from "react-dom/client";
import { BrowserRouter } from "react-router-dom";

import "./index.css"
import App from "./App";

ReactDOM.createRoot(document.getElementById("root") as HTMLElement).render(
    <React.StrictMode>
        <BrowserRouter basename={"/admin"}>
            <App/>
        </BrowserRouter>
    </React.StrictMode>
)
```
