### Esempio FireCMS self-hosted

Per FireCMS self-hosted, puoi definire le tue viste di primo livello personalizzate aggiungendole nel tuo componente `useBuildNavigationController`.
Puoi passarle come array di oggetti `CMSView` o una funzione callback che restituisce un array di oggetti `CMSView`.

```tsx
import {useMemo} from "react";

        path: "additional",
        name: "Vista aggiuntiva",

        // PuĂ˛ essere qualsiasi componente React
        view: <ExampleCMSView/>

        views: ({ user }) =>{
            if(!user) {
                return [];
            }
            return customViews;
        },
        collections,
        authController,
        dataSourceDelegate: firestoreDelegate
    });
```
