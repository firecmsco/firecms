Quando si costruisce un'interfaccia utente, è spesso necessario includere icone per rappresentare diverse azioni, stati o contenuti.
FireCMS utilizza le **icone Material Design**, disponibili gratuitamente da Google. Queste icone sono comodamente
raggruppate come componenti React indipendenti, che possono essere utilizzate nei propri componenti.

In alcuni casi potresti aver bisogno di usare le icone come componenti, ad esempio in un pulsante. In altri casi, potresti
dover specificare la chiave dell'icona.

In questa vista puoi vedere tutte le icone disponibili e le loro chiavi. Basta **cercare** l'icona di cui hai bisogno!
