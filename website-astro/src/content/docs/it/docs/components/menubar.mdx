Il componente Menubar fornisce una barra menu personalizzabile e accessibile, costruita usando le primitive Radix UI Menubar. Può essere usato per creare strutture di menu complesse con varie funzionalità come sottomenu, checkbox e gruppi radio.

## Utilizzo

Per usare il `Menubar`, importalo dai tuoi componenti e componilo con altri componenti Menubar correlati come `MenubarMenu`, `MenubarTrigger`, `MenubarContent`, `MenubarItem`, ecc.
