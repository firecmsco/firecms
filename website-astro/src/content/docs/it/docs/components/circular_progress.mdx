Il componente `CircularProgress` è usato per indicare stati di caricamento all'utente. Mostra un indicatore rotante che può essere personalizzato in dimensione.

## Utilizzo

Per incorporare il componente `CircularProgress`, importalo dalla tua libreria di componenti e opzionalmente passa le prop `size` e `className` per la personalizzazione.
