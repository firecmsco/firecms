I separatori sono usati per distinguere visivamente il contenuto in un layout o lista. Possono essere orizzontali o verticali, rendendoli versatili per varie esigenze di design.

## Utilizzo

Per usare il `Separator`, importalo dai tuoi componenti e specifica l'`orientation` e, opzionalmente, la prop `decorative` per controllarne l'aspetto.
