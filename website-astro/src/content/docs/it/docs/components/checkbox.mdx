I Checkbox sono usati per la selezione tra più opzioni. Possono alternare tra selezionato, deselezionato e uno stato intermedio.

## Utilizzo

Per usare il `Checkbox`, importalo dai tuoi componenti e passa `checked`, `onCheckedChange` e, opzionalmente, le prop `disabled`, `size` e `color`.
