Las pestañas (Tabs) se utilizan para la navegación entre diferentes vistas o secciones dentro del mismo contexto.

## Uso

Para usar el componente `Tabs`, impórtalo desde tus componentes junto con los componentes secundarios `Tab` y pasa las props requeridas.
