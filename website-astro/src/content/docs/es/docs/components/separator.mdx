Los separadores se utilizan para distinguir visualmente el contenido en un diseño o lista. Pueden ser horizontales o verticales, lo que los hace versátiles para diversas necesidades de diseño.

## Uso

Para usar `Separator`, impórtalo desde tus componentes y especifica la orientación (`orientation`) y, opcionalmente, la prop `decorative` (decorativo) para controlar su apariencia.
