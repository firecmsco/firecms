:::tip
FireCMS Cloud es el **panel de administración de Firebase y CMS headless definitivo**.
Proporciona un **back-office basado en React** totalmente administrado que le permite administrar sus datos, usuarios y contenido de Firestore con facilidad.

**Listo para no code**: Visualice y edite datos inmediatamente.
**Habilitado para código**: Suba código React personalizado, vistas de entidad y constructores de propiedades.
:::

## ¿Por qué FireCMS Cloud?

Crear un **panel de administración** o una **herramienta interna** personalizados desde cero requiere mucho tiempo. FireCMS Cloud le brinda un **CMS** listo para producción en minutos, adaptado para **Firebase** y **Firestore**.

### Características principales

- **Servicio administrado**: Nosotros nos encargamos del alojamiento, la escalabilidad y las actualizaciones. Usted se enfoca en sus datos y lógica de negocios.
- **Hermosa interfaz de usuario**: Una interfaz moderna y receptiva para su **base de datos Firestore**.
- **Editor de colecciones**: Defina su esquema visualmente con nuestro editor sin código.
- **Fácil para desarrolladores**: Extienda con **React**. Cree campos de formulario personalizados, widgets de panel de control y vistas previas.
- **Acceso basado en roles**: Control de permisos granular para su equipo de contenido y administradores.
