Las propiedades (properties) definen cada **campo** en un formulario o columna en una colección incluida en una
colección de entidades.

Puedes construir propiedades creando el objeto directamente o usando
el método de ayuda (helper method) `buildProperty` (simplemente la función de identidad que usa
el sistema de tipos de Typescript para validar la entrada).

Es posible que también desees actualizar las propiedades **dinámicamente**, según el entityId, la
ruta (path) o los valores actuales. Consulta
la [sección de campos condicionales](conditional_fields).

Consulta los diferentes campos disponibles en la sección de campos, o las
diferentes configuraciones de propiedades.
