Al crear una interfaz de usuario, a menudo es necesario incluir iconos para representar diferentes acciones, estados o contenidos.
FireCMS utiliza **iconos de Material Design**, que están disponibles de forma gratuita en Google. Estos iconos están empaquetados
convenientemente como componentes React independientes, que puedes usar en tus propios componentes.

En algunos casos podrías necesitar usar los iconos como componentes, por ejemplo en un botón. En otros casos, podrías
necesitar especificar la clave del icono.

En esta vista puedes ver todos los iconos disponibles y sus claves. ¡Simplemente **busca** el icono que necesites!
