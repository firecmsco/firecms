:::important
FireCMS es un **framework de CMS headless y panel de administración de código abierto** construido con React y TypeScript. Se conecta directamente a tu proyecto de **Firebase** o **MongoDB** para proporcionar una interfaz potente, intuitiva y con seguridad de tipos para administrar tus datos.
:::

FireCMS está diseñado para ser la forma más rápida de construir un back-office listo para producción. Genera vistas completas CRUD (Crear, Leer, Actualizar, Borrar) a partir de la configuración de datos, no impone **restricciones de estructura de datos** y se puede configurar en minutos.

## Dos formas de empezar

El primer paso más importante es decidir qué versión de FireCMS es la adecuada para ti.

### FireCMS Cloud

Para velocidad y conveniencia. Una solución totalmente administrada y sin mantenimiento con una prueba gratuita de 1 mes.

  * ✅ **Empieza en 60 segundos**, no requiere programación.
  * ✅ Plataforma totalmente administrada con actualizaciones automáticas.
  * ✅ Editor de esquemas sin código e inferencia de datos.
  * ✅ Personalización low-code con componentes de React.

[**Iniciar prueba gratuita**](https://app.firecms.co)

### Self-Hosted

Para máximo control y personalización. Gratuito, de código abierto y desplegado en tu propio entorno.

  * ✅ **Gratis y Open Source** (Licencia MIT).
  * ✅ Control total sobre tu código y despliegue.
  * ✅ Personalización profunda con React y TypeScript.
  * ✅ `npx create-firecms-app` para comenzar.

[**Inicio rápido Self-Hosted**](/docs/self)

## ¿Qué puedes construir?

FireCMS es lo suficientemente flexible como para ser el backend de cualquier proyecto. Los casos de uso comunes incluyen:

  * **Paneles de administración personalizados:** Crea una interfaz de usuario con seguridad de tipos para administrar los datos de tu aplicación (por ejemplo, usuarios, productos, pedidos) sin las molestias del frontend.
  * **Herramientas internas:** Crea herramientas personalizadas para que tus equipos de operaciones, ventas o soporte administren flujos de trabajo, rastreen datos y ejecuten procesos internos.
  * **CMS Headless:** Potencia tus sitios web, aplicaciones móviles y contenido de marketing desde una única y potente interfaz de administración.

## Funciones principales para desarrolladores

FireCMS está construido por desarrolladores, para desarrolladores.

### 1\. Esquema como código

Define tus modelos de datos como objetos de TypeScript. Tu código se convierte en la única fuente de la verdad, permitiendo control de versiones, seguridad de tipos y un flujo de trabajo basado en git sin problemas.

### 2\. UI CRUD instantánea

FireCMS lee tu esquema y genera automáticamente una interfaz CRUD completa, de alto rendimiento y hermosa con tablas, formularios, validación y más.

### 3\. Extensibilidad radical

Si puedes construirlo en React, puedes construirlo en FireCMS. Crea campos de formulario personalizados, vistas de nivel superior completamente nuevas o reemplaza cualquier parte de la interfaz de usuario con tus propios componentes. Sin cajas negras.
