:::tip
O FireCMS Cloud é o **painel de administração Firebase definitivo** e **headless CMS**.
Ele fornece um **back-office baseado em React** totalmente gerenciado que permite gerenciar seus dados, usuários e conteúdo Firestore com facilidade.

**Pronto para uso sem código**: Visualize e edite dados imediatamente.
**Habilitado para código**: Faça upload de código React personalizado, views de entidade e property builders.
:::

## Por que o FireCMS Cloud?

Construir um **painel de administração** ou uma **ferramenta interna** personalizada do zero é demorado. O FireCMS Cloud oferece um **CMS** pronto para produção em minutos, personalizado para **Firebase** e **Firestore**.

### Recursos Principais

- **Serviço Gerenciado**: Cuidamos de hospedagem, escalabilidade e atualizações. Você foca em seus dados e lógica de negócios.
- **UI Bonita**: Uma interface moderna e responsiva para o seu **banco de dados Firestore**.
- **Editor de Coleções**: Defina seu schema visualmente com nosso editor sem código.
- **Amigável para Desenvolvedores**: Estenda com **React**. Crie campos de formulário, widgets de dashboard e prévias personalizadas.
- **Acesso Baseado em Papéis**: Controle de permissões granular para sua equipe de conteúdo e administradores.
