:::important
O FireCMS é um **framework CMS headless e painel de administração open-source** construído com React e TypeScript. Ele se conecta diretamente ao seu projeto **Firebase** ou **MongoDB** para fornecer uma interface poderosa, intuitiva e com type-safety para gerenciar seus dados.
:::

O FireCMS foi projetado para ser a maneira mais rápida de criar um back-office pronto para produção. Ele gera views CRUD completas (Create, Read, Update, Delete) a partir da sua configuração de dados, **não impõe restrições na estrutura dos dados** e pode ser configurado em minutos.

## Duas Formas de Começar

O primeiro passo mais importante é decidir qual versão do FireCMS é certa para você.

### FireCMS Cloud

Para velocidade e conveniência. Uma solução totalmente gerenciada, sem manutenção, com um mês de teste gratuito.

  * ✅ **Comece em 60 segundos**, sem necessidade de programação.
  * ✅ Plataforma totalmente gerenciada com atualizações automáticas.
  * ✅ Editor de schema no-code e inferência de dados.
  * ✅ Personalização low-code com componentes React.

[**Iniciar Teste Gratuito**](https://app.firecms.co)

### Self-Hosted

Para máximo controle e personalização. Gratuito, open-source e executado no seu próprio ambiente.

  * ✅ **Gratuito e Open Source** (Licença MIT).
  * ✅ Controle total sobre seu código e deployment.
  * ✅ Personalização profunda com React e TypeScript.
  * ✅ `npx create-firecms-app` para começar.

[**Quickstart Self-Hosted**](/docs/self)

## O que você pode construir?

O FireCMS é flexível o suficiente para ser o backend de qualquer projeto. Os casos de uso comuns incluem:

  * **Painéis de Administração Personalizados:** Crie uma UI com type-safety para gerenciar os dados da sua aplicação (ex.: usuários, produtos, pedidos) sem o trabalho no frontend.
  * **Ferramentas Internas:** Construa ferramentas sob medida para suas equipes de operações, vendas ou suporte para gerenciar fluxos de trabalho, rastrear dados e executar processos internos.
  * **CMS Headless:** Alimente seus sites, aplicativos mobile e conteúdo de marketing a partir de uma única e poderosa interface de administração.

## Funcionalidades Principais para Desenvolvedores

O FireCMS é construído por desenvolvedores, para desenvolvedores.

### 1\. Schema como Código

Defina seus modelos de dados como objetos TypeScript. Seu código se torna a única fonte da verdade, habilitando controle de versão, type-safety e um fluxo de trabalho baseado em git sem interrupções.

### 2\. UI CRUD Instantânea

O FireCMS lê seu schema e gera automaticamente uma interface CRUD completa, performática e bonita com tabelas, formulários, validação e muito mais.

### 3\. Extensibilidade Radical

Se você pode construí-lo em React, pode construí-lo no FireCMS. Crie campos de formulário personalizados, views de nível superior completamente novas, ou substitua qualquer parte da interface com seus próprios componentes. Sem caixas pretas.
