Separadores são usados para distinguir visualmente o conteúdo em um layout ou lista. Podem ser horizontais ou verticais, tornando-os versáteis para várias necessidades de design.

## Uso

Para usar o `Separator`, importe-o dos seus componentes e especifique a `orientation` e, opcionalmente, a prop `decorative` para controlar sua aparência.
