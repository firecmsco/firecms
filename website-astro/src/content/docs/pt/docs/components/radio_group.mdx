Radio Group permite que os usuários selecionem uma opção de um conjunto. É útil para cenários de seleção exclusiva onde apenas uma escolha é permitida.
Cada item do grupo de rádio é representado por um componente `RadioGroupItem` e tipicamente envolvido por um componente `Label`.

## Uso

Para usar o `RadioGroup` e o `RadioGroupItem`, importe-os dos seus componentes e estruture suas opções usando o `RadioGroup` como contêiner e o `RadioGroupItem` para cada opção.
