O componente Menubar fornece uma barra de menu personalizável e acessível, construída usando primitivas Radix UI Menubar. Pode ser usado para criar estruturas de menu complexas com várias funcionalidades como submenus, checkboxes e grupos de rádio.

## Uso

Para usar o `Menubar`, importe-o dos seus componentes e componha-o com outros componentes relacionados ao Menubar como `MenubarMenu`, `MenubarTrigger`, `MenubarContent`, `MenubarItem`, etc.
