O componente `CircularProgress` é usado para indicar estados de carregamento ao usuário. Exibe um indicador giratório que pode ser personalizado em tamanho.

## Uso

Para incorporar o componente `CircularProgress`, importe-o da sua biblioteca de componentes e, opcionalmente, passe as props `size` e `className` para personalização.
