O componente `Sheet` é usado para exibir painéis deslizantes das bordas da tela. Pode ser aberto do topo, inferior, esquerda ou direita, e pode ser opaco ou transparente.

## Uso

Para usar o `Sheet`, importe-o dos seus componentes e passe as props `open`, `side`, `transparent` e `onOpenChange`.
