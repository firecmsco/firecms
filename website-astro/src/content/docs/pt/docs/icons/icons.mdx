Ao criar uma interface de usuário, muitas vezes é necessário incluir ícones para representar diferentes ações, estados ou conteúdos.
O FireCMS utiliza **ícones do Material Design**, disponíveis gratuitamente pelo Google. Esses ícones estão empacotados
convenientemente como componentes React independentes, que podem ser usados nos seus próprios componentes.

Em alguns casos, você pode precisar usar os ícones como componentes, por exemplo, em um botão. Em outros casos, pode ser necessário
especificar a chave do ícone.

Nesta visualização, você pode ver todos os ícones disponíveis e suas chaves. Basta **pesquisar** o ícone que você precisa!
