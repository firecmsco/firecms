Neste tutorial vamos mostrar como você pode adicionar um botão à sua coleção,
que permitirá **copiar entidades de outra coleção**.

É muito comum em bancos de dados NoSQL, como o Firestore, manter dados desnormalizados
em diferentes coleções. Ou seja, manter as mesmas informações em múltiplos
locais, em vez de bancos de dados normalizados onde as informações devem ser idealmente
armazenadas apenas uma vez.
