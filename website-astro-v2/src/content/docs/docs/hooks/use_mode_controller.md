Hook to retrieve the current mode (`light` | `dark` | `system`), and `setMode`.

Consider that in order to use this hook you need to have a parent
`FireCMS` component in the tree.

### Props:

```tsx
{
    mode: "light" | "dark";
    setMode: (mode: "light" | "dark") => void;
}
```
